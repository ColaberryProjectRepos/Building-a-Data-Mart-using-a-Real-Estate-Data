Step-1 Create ColaberryResortRealty Database in SQL Server Management Studio 1

Open the SSMS window. Right click on database then click on “New Database”.

Now, we can see New Database screen window, so we need to write database name then click ok.



Now, we can see our Database “ColaberryResortRealty”.




Use below link to see how to create the Database in SQL Server Management Studio:
